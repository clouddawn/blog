# 质数练习的优化

原代码：

```javascript
for (var i = 2; i <= 10000; i++) {
  var zhi = true;
  for (var j = 2; j < i; j++) {
    if (i % j == 0) {
      zhi = false;
    }
  }
  if (zhi) {
    console.log(i);
  }
}
```

优化：

```javascript
for (var i = 2; i <= 10000; i++) {
  var zhi = true;
  for (var j = 2; j < Math.sqrt(i); j++) {
    if (i % j == 0) {
      zhi = false;
      break;
    }
  }
  if (zhi) {
    console.log(i);
  }
}
```
