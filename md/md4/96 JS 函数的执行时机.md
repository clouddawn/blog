# JS 函数的执行时机

## 为什么如下代码会打印 6 个 6 ？

```js
let i = 0
for(i = 0; i<6; i++){
  setTimeout(()=>{
    console.log(i)
  },0)
}
```

因为setTimeout会等到当前代码的for循环执行完了，再去执行console.log(i) 

而for循环执行完之后，i已经是6了，所以会打印出6个6

## 让上面代码打印 0、1、2、3、4、5 的方法

```js
for(let i = 0; i<6; i++){
  setTimeout(()=>{
    console.log(i)
  },0)
}
```
