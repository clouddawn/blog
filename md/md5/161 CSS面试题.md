5. 清除浮动说一下

   背代码

   ```css
    .clearfix:after{
        content: '';
        display: block; /*或者 table*/
        clear: both;
    }
    .clearfix{ // 加到容器上，里面的子元素的浮动就被清除了
        zoom: 1; /* IE 兼容*/
    }
   ```
