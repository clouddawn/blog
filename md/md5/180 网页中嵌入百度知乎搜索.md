# 如何在自己的网页中嵌入百度、知乎搜索？

```html
<form class="searchForm" action="https://www.baidu.com/s">
    <input type="text" name="wd">
    <button>百度搜索</button>
</form>
<form class="searchForm" action="https://www.zhihu.com/search?type=content">
    <input type="text" name="q">
    <button>知乎搜索</button>
</form>
```
