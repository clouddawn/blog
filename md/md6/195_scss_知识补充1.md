## 变量（$）

```scss
$font-color: rgb(251, 209, 0);
p {
  color:$font-color;
}
```

-

```scss
/*color.scss*/
$theme-color: #22a6f2;
```

.

```scss
@import 'color.scss';
p {
  color: $theme-color;
}
```

## darken()

```scss
darken($color,$amount);
```

* 使 $color 颜色加深
* $amount 必须是介于 0% 和 100% (含) 之间的数字，将 $color  的 HSL 亮度降低 $amount。

```scss
p {
  color:darken($theme-color,30%);
}
```

## 父选择器（&）

在嵌套 CSS 规则时，有时也需要直接使用嵌套外层的父选择器，例如，当给某个元素设定 `hover` 样式时。

```scss
a {
  font-weight:bold;
  &:hover {
    color:red;
  }
}
```
