# 神经网络

神经网络是一种受人脑神经元结构启发的数学模型，它由大量简单的计算单元（称为神经元或节点）和它们之间的连接组成，这些神经元通常被组织成一层一层的结构。

在神经网络里，参数主要是权重和偏置。

* 权重：代表一个神经元连接到下一个神经元时，信号的强弱。就像音量旋钮，可以放大或减弱前一个神经元的影响。
* 偏置：可以理解一个神经元自身的敏感度或激活门槛，它决定了神经元是否容易被触发。

一个训练好的神经网络，本质上就是一个极其复杂的数学函数，它能将输入（比如一张图片的像素）映射到期望的输出（比如猫或狗的标签）。



举个例子：识别手写数字8

1. 输入层：接收图片的每个像素的灰度值（比如784个神经元对应28x28像素）。
2. 隐藏层：中间的若干层神经元。第一层可能识别线条和边缘；第二层可能将线条组合成圆圈或弧线；更深层则能组合成8的两个环。每一层的计算都用到了权重（决定哪些边缘重要）和偏置（决定是否激活某个特征探测器）。
3. 输出层：有10个神经元（代表0-9），哪个神经元的输出值最大，模型就预测是哪个数字。



神经网络通过不断尝试、计算误差、调整参数来学习。这个调整过程有一个专门的名字，叫反向传播，它配合优化器（如梯度下降算法）来工作。

可以这样理解：网络做出预测 → 计算预测与真实答案的误差 → 反向传播这个误差信号，告诉每个权重和偏置：你们应该往大调还是往小调，才能让下次预测更准 → 调整参数 → 重复。

这就像一个拥有亿万个可调旋钮的复杂机器，通过反复实验，把每个旋钮都拧到最佳位置。



总结一下：神经网络就是一类结构呈层级化、参数包含权重和偏置的数学模型。它能通过学习，自动从海量数据中提取从简单到复杂的特征，最终拟合出一个极其强大的非线性函数，来解决分类、回归等任务。
