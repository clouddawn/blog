# scikit-learn 库

`scikit-learn` （常简写为 `sklearn`）是 Python 生态中最流行、最成熟的机器学习库之一，主要面向传统机器学习。它构建在 NunPy、SciPy 和 matplotlib 之上，提供了大量高效、易用的工具，覆盖了从数据预处理到模型评估的完整流程。

## 核心设计理念

* **统一的 API 接口：**几乎所有模型都遵循“导入→实例化→`fit()`训练→`predict()`/`transform()`应用 ”的模式，学习成本低。
* **专注传统机器学习：**擅长处理中小规模的结构化数据（表格数据），不涉及深度学习、强化学习或图像/语音的原始信号处理。
* **高性能与可解释性：**底层算法多采用 Cython/C 实现，同时提供特征重要性、系数等易于解释的属性。

## 主要优势

* **开箱即用，算法丰富：**几乎涵盖所有经典机器学习算法，且实现稳定、经过充分测试。
* **与 Python 科学计算栈无缝集成：**数据格式兼容 NumPy 数组和 pandas DataFrame，可直接对接 matplotlib、seaborn 进行可视化。
* **生产可行性：**模型可序列化（`joblib` 和 `pickle`）并部署到生产环境。

## 适用边界

* **不适合深度学习：**没有神经网络复杂架构，如需深度学习应使用 PyTorch、TensorFlow 等。
* **对超大数据集支持有限：**算法多要求数据能装入内存，不能直接处理流式数据或分布式训练（虽然有 `partial_fix` 等增量学习接口，但覆盖不全）。
* **非结构化数据需先行处理：** 图像、文本、音频等需要先由其他库提取为特征向量，再交给 scikit-learn 建模。
