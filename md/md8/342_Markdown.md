# Markdown

Markdown 是一种轻量级的**标记语言**，它的核心设计理念是 **“易读易写”**——让你用最自然的纯文本格式（如 `#` 表示标题，`*` 表示列表）来撰写文档，并能快速转换成结构化的 HTML 或其他格式。

------

## 1. Markdown 的核心语法

### 标题

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```



### 文本样式

```
*斜体*  或 _斜体_
**粗体** 或 __粗体__
***粗斜体***
~~删除线~~
`行内代码`
```



### 列表

**无序列表**（使用 `-`、`+` 或 `*`）：

```
- 苹果
- 香蕉
  - 子项（缩进2或4空格）
- 橙子
```



**有序列表**（数字 + 点）：

```
1. 第一步
2. 第二步
   1. 子步骤（缩进）
3. 第三步
```



### 链接与图片

markdown

```
[显示文本](https://example.com "可选标题")

![替代文本](图片URL "可选标题")
```



### 引用

```
> 这是一段引用。
> 可以多行。
>> 嵌套引用
```



### 代码块

```
​```python
def hello():
    print("Hello, Markdown!")
​```
```



### 表格

```
| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 苹果   | 红色     | 5元    |
| 香蕉   | 黄色     | 3元    |
```



### 任务列表（GitHub 扩展）

```
- [x] 已完成任务
- [ ] 待办任务
```



### 转义字符

使用反斜线 `\` 转义 Markdown 符号，例如 `\*` 显示字面星号。



------

## 2. 示例：一段 Markdown 写成的短文档

```
# 我的购物清单

> 周末去超市采购。

## 食品
- [x] 牛奶
- [ ] 面包
- [ ] 鸡蛋

## 非食品
1. 垃圾袋
2. 洗洁精

| 项目 | 预算 |
|------|------|
| 食品 | 100元 |
| 非食品 | 50元 |

代码示例（计算总预算）：
​```python
total = 100 + 50
print(f"总预算：{total}元")
```
