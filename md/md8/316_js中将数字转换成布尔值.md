# js中数字转换为布尔值

## 1. 使用 `Boolean()` 函数

### 基本用法

```javascript
// 整数转换
console.log(Boolean(0));      // false
console.log(Boolean(1));      // true
console.log(Boolean(-1));     // true

// 浮点数转换
console.log(Boolean(0.0));    // false
console.log(Boolean(0.1));    // true
console.log(Boolean(-3.14));  // true
console.log(Boolean(NaN));        // false ⚠️ 注意：NaN 是 false
```



## 2. 双重否定运算符 `!!`

这是最常用的快速转换方法：

```javascript
// 使用 !! 转换为布尔值
console.log(!!0);         // false
console.log(!!1);         // true
console.log(!!-1);        // true
```



## 3. 隐式转换（自动转换）

### 在条件语句中

```javascript
// if 语句
const num = 10;
if (num) {  // 隐式转换为布尔值
    console.log("数字非零");  // 会执行
}

const zero = 0;
if (zero) {
    console.log("这行不会执行");
}

// 三元运算符
const value = 5;
const result = value ? "真值" : "假值";
console.log(result);  // "真值"
```



### 在逻辑运算中

```javascript
// 逻辑与 (&&) - 返回第一个假值或最后一个真值
console.log(5 && 10);   // 10
console.log(0 && 10);   // 0
console.log(1 && 2 && 3);  // 3
console.log(1 && 0 && 3);  // 0

// 逻辑或 (||) - 返回第一个真值或最后一个假值
console.log(5 || 10);   // 5
console.log(0 || 10);   // 10
console.log(0 || 0 || 3);  // 3
console.log(0 || null || undefined);  // undefined

// 空值合并运算符 (??) - ES2020
console.log(0 ?? 10);   // 0（只过滤null和undefined）
console.log(null ?? 10); // 10
console.log(undefined ?? 10); // 10
```



## 4. 数字的"真假值"规则

JavaScript 的假值列表（转换为布尔值为 false 的值）：

```javascript
const falsyValues = [
    0,
    -0,
    NaN,
    "",      // 空字符串
    false,
    null,
    undefined
];

console.log("JavaScript 假值列表:");
falsyValues.forEach(value => {
    console.log(`${JSON.stringify(value)} -> ${Boolean(value)}`);
});
```
