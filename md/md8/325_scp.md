# 什么是 SCP ？

`scp` 是 Linux/Unix 系统中基于 SSH 协议的安全远程文件复制工具。它可以在本地与远程主机之间，或两台远程主机之间加密传输文件。

## 工作原理

`scp` 底层使用 SSH 进行数据传输和身份验证，因此传输过程是加密的，和 `ssh` 一样安全。它相当于在主机间通过 SSH 隧道复制文件。

## 基本语法

```bash
scp [选项] 源文件 目标路径
```

* 源文件和目标路径可以是本地路径，也可以是 `用户名@主机:路径` 的远程路径。
* 复制方向取决于参数位置：
  * 从本地到远程：`scp 本地文件 用户@主机:远程路径`
  * 从远程到本地：`scp 用户@主机:远程文件 本地路径 `
  * 两台远程主机之间：`scp 用户A@主机A:文件 用户B@主机B:路径`

## 常用选项

| 选项          | 说明                                                       |
| ------------- | ---------------------------------------------------------- |
| `-P 端口`     | 指定远程主机的 SSH 端口（大写 P ，区别与 ssh 的小写 `-p`） |
| `-r`          | 递归复制整个目录                                           |
| `-p`          | 保留原文件的修改时间、访问时间和权限（小写 p）             |
| `-c`          | 传输时开启压缩，可加快速度                                 |
| `-i 密钥文件` | 指定身份验证的私钥文件（如 `~/.ssh/id_rsa`）               |
| `-q`          | 静默模式，不显示传输进度和诊断信息                         |
| `-v`          | 详细模式，输出调试信息，方便排查问题                       |

## 示例

* 上传本地文件到远程

  ```bash
  scp -P 1435 ./file.tar user@10.34.0.171:/home/user/
  ```

* 从远程下载文件到本地

  ```scp
  scp user@192.168.1.100:var/log/app.log ./logs/
  ```
