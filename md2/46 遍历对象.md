# 遍历对象

```javascript
        var obj = {
            name:'王胖子',
            gender:'男',
            weight: 185,
            height: 180
        }
```
```javascript
        for(var key in obj){
            console.log(key); // key 变量 输出 得到的是 属性名
        }
```
![image](../images/46/1.png)(1)

```javascript
        for(var key in obj){
            console.log(obj[key]); // obj[key] 得到的是 属性值
        }
```
![image](../images/46/2.png)(2)

```javascript
        for(var key in obj){
            console.log(key + '：' + obj[key]);
        }
```
![image](../images/46/3.png)(3)

* 变量名 key 或 k 是习惯用法，不是硬性规定
