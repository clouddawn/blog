# 作用域

## 作用域概述

* 通常来说，一段程序代码中所用到的名字并不总是有效和可用的，而限定这个名字的**可用性的代码范围**就是这个名字的**作用域**。  

* 作用域的使用提高了程序逻辑的局部性，增强了程序的可靠性，减少了名字冲突。  

## JS的作用域(ES6之前)

### 全局作用域  
* 整个 script 标签，或者是一个单独的 JS 文件。

```javascript
        var num = 10;
        console.log(num);
```
### 局部作用域(函数作用域)
* 在函数内部就是局部作用域，这个代码的名字只在函数内部起效果和作用。

```javascript
        var num = 10;
        function fn(){
            var num = 20;
            console.log(num);
        }
        fn();
        console.log(num);
```
